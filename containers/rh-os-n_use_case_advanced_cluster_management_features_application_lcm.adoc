== 应用程序生命周期管理

要创建应用程序并在一组集群中对其进行管理，

. 从边栏导航到管理应用程序，然后单击创建应用程序。提供要创建的应用程序的详细信息，然后单击保存。
+
image::redhat_openshift_image78.jpg[创建应用程序]

. 安装应用程序组件后，此应用程序将显示在列表中。
+
image::redhat_openshift_image79.jpg[应用程序列表]

. 现在，可以从控制台监控和管理此应用程序。
