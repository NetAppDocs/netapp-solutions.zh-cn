== 前提条件

* NetApp ONTAP 集群。
* Red Hat OpenShift 集群
* 集群上安装的 Trident 。
* 安装了 tridentctl 和 oc 工具并将其添加到 $path 中的管理工作站。
* 对 ONTAP 的管理员访问权限。
* 对 OpenShift 集群的集群管理员访问。
* 集群已与身份提供程序集成。
* 身份提供程序经过配置，可以有效区分不同团队中的用户。
